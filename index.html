Shields against abuse of anonymity

 

Employee Console

Sender(s) is (are) blocked for 3 months
and HR is alerted.

HR must be informed.

Serial anonymized Feedback Sending

If an employee sends 3 feedbacks to a colleague in 2 months, he/she is either abusing or abused.

Coordinated anonymized Feedback Sending

If more than 3 employees send feedbacks to the same colleague in 2 months, either they are abusing or being abused.

HR Console

Abuse of anonymized report to HR

if using Inquiry tools, HR finds an employee’s report to be repeatedly unsubstantiated, he/she can suspend this employee’s access to anonymity. Employee can still use the non-anonymous complaint to HR feature.
